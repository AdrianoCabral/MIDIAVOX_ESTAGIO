Linguagens/plataformas utilizadas em cada componente:

CLIENTE: javascript/html/css/jquery

WEBSERVICE: node.js, express

BANCO DE DADOS: banco mysql

Instalações:

NODE JS:
windows: https://nodejs.org/en/download/

linux: sudo apt install nodejs

MYSQL:
windows:https://dev.mysql.com/downloads/mysql/

linux: sudo apt install mysql-server

para execução do serviço:

executar o script da criação do banco no mysql workbench ou pelo terminal do mysql server

abrir terminal no diretório do projeto, e digitar os comandos


$ npm install

$ node index.js

acessar localhost:3000 no seu browser
